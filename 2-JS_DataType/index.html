JavaScript has 8 Datatypes

Primitive Data Types :

1. String - Sequence of characters.Must be in quotes or backticks
2. Number -Intgers as well as floating-point numbers
3. Bigint -Numbers that are greater than the "Number" type can handle.
4. Boolean -Logical entity/ true or false
5. Undefined - A Variable that has not yet been defined / assigned.
6. Null -Idential absence of any Object value.
7. Symbol -Built-in object whose constructor retturns a unique symbol.
Note:
Primitive Values are stored on the Stack.

Reference Types(Objects)

8. Object

The Object Datatype The object data type can contain:
1. An object
2. An array
3. A date
4. A Functions
Note:
Reference Values are stored on the Heap! And Accessed by Reference.

Note:
A JavaScript variable can hold any type of data.
